# ls

> [!info] 功能
> 列出目录内容。

## 语法

- `ls`
- `ls -l`
- `ls -a`
- `ls -lh`
- `ls -R`
- `ls -ld dir`

## 输入参数

- `-l`：长格式，显示权限、链接数、属主、属组、大小、时间。
- `-a`：显示隐藏文件。
- `-d`：只显示目录本身属性，不进入目录内部。
- `-R`：递归查看子目录。
- `-h`：人类可读大小显示。

## 输出

- 目录中的文件和子目录列表。

## 常见用法

- 看目录里有什么。
- 配合 `-l` 检查权限和元信息。

## 易错点

- `ls dirname` 和 `ls -ld dirname` 不是一回事。
