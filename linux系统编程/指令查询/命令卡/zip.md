# zip

> [!info] 功能
> 创建 zip 压缩包。

## 语法

- `zip a.zip file`
- `zip -r a.zip dir`

## 输入参数

- `a.zip`：输出压缩包。
- `file` / `dir`：要压缩的对象。
- `-r`：递归压缩目录。

## 输出

- zip 压缩包。

## 常见用法

- 跨平台传文件。

## 易错点

- 压缩目录通常要加 `-r`。
