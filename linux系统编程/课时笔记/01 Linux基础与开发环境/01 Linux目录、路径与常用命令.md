# Linux目录、路径与常用命令

## 本节学什么

- Linux 常见目录含义
- 绝对路径与相对路径
- shell 常用快捷键
- 基础命令入口怎么查：[[pwd]]、[[cd]]、[[which]]、[[man]]、[[history]]、[[alias]]、[[date]]




## 本节学什么详解

- Linux 常见目录含义：Linux 的目录不是随便摆放文件的文件夹，而是一套有固定职责的树状结构；先认清 `/`、`/home`、`/etc`、`/proc` 这些常见目录，后面你看命令和系统调用时才知道数据到底放在哪里。
- 绝对路径与相对路径：绝对路径从根目录 `/` 开始，能够唯一定位文件；相对路径依赖当前工作目录，写起来更短，但前提是你必须知道自己现在在哪个目录里。
- shell 常用快捷键：这些快捷键不是“背口诀”，而是为了更高效地操作终端：补全命令、快速移动光标、删除字符、终止当前命令，都会直接影响你后面写代码和调试的效率。
- 基础命令入口怎么查：[[pwd]]、[[cd]]、[[which]]、[[man]]、[[history]]、[[alias]]、[[date]]：这一组命令分别解决“看当前在哪、怎么切目录、这个命令在哪、命令手册在哪里、历史命令怎么找、别名怎么配、当前时间怎么看”这几个最基础的问题。

## 知识点补充
- 绝对路径从 `/` 开始，能够直接定位到文件系统中的唯一位置。
- 相对路径依赖当前工作目录，适合在目录树中连续切换。
- `pwd`、`cd`、`which`、`man` 是后面所有学习的入口命令。

## 内容速览

- 根目录 `/` 是整个文件系统树的起点。
- `/bin`、`/sbin`、`/etc`、`/home`、`/root`、`/proc` 是高频目录。
- 绝对路径以 `/` 开始，相对路径相对当前工作目录。
- `Tab` 补全、`Ctrl+C` 中断、`Ctrl+A/E` 移动光标是高频操作。

- `/proc`、`/dev`、`/etc`、`/home`、`/root` 这几个目录在课程里经常重复出现。
- `pwd`、`cd`、`which`、`man` 这些命令是后面所有学习的入口。

- `Tab` 补全、历史命令、光标移动、删除字符，都会直接影响后续编程效率。
