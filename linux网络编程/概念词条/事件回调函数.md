# 事件回调函数

## 它是什么

- 事件回调函数是事件满足条件时由 Libevent 自动调用的函数。
- 程序员把处理逻辑写在回调函数里，而不是手动在循环里判断每个 fd。

## 普通 event 回调原型

```c
typedef void (*event_callback_fn)(evutil_socket_t fd, short what, void *arg);
```

## 参数理解

- `fd`：触发事件的文件描述符。
- `what`：触发的事件类型，例如 `EV_READ`、`EV_WRITE`。
- `arg`：创建事件时传入的用户参数。

## 易错点

- 回调函数不要长时间阻塞，否则会拖慢整个事件循环。
- `arg` 指向的对象要保证在回调触发时仍然有效。
