# ping

## 命令作用

- `ping` 用来测试本机到目标主机之间的基本网络连通性。
- 它通常基于 ICMP 协议发送探测报文，并等待对端响应。

## 常见写法

```bash
ping 127.0.0.1
ping 192.168.1.1
ping www.baidu.com
ping -c 4 127.0.0.1
```

## 参数说明

- `目标地址`：可以是 IP 地址，也可以是域名。
- `-c 次数`：指定发送多少次探测报文后停止。

## 常见用途

- 测试回环地址 `127.0.0.1` 是否正常。
- 测试虚拟机、宿主机、服务器之间是否能互通。
- 在运行 socket 程序前，先确认网络链路是否基本可达。

## 注意点

- `ping` 成功只能说明网络层基本可达，不代表 TCP 端口一定开放。
- `ping` 失败也不一定代表主机不存在，可能是防火墙或目标主机禁用了 ICMP 响应。
