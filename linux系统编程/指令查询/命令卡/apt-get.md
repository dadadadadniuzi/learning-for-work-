# apt-get

> [!info] 功能
> Debian / Ubuntu 系列的软件包管理命令。

## 语法

- `sudo apt-get update`
- `sudo apt-get install pkg`
- `sudo apt-get remove pkg`

## 输入参数

- `update`：刷新软件列表。
- `install`：安装软件包。
- `remove`：卸载软件包。
- `pkg`：软件包名。

## 输出

- 完成软件列表刷新、安装或卸载。

## 常见用法

- 装课程里需要的工具。

## 易错点

- `update` 只是刷新索引，不是升级软件。
