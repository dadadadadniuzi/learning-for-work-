# Makefile规则

## 核心三要素

- 目标：要生成什么文件。
- 依赖：目标依赖哪些文件。
- 命令：满足条件后要执行什么命令。

## 规则格式

```makefile
target: dep1 dep2
	command
```

## 自动变量

- `$@`：目标名
- `$<`：第一个依赖
- `$^`：所有依赖

## 伪目标

- 常见如 `clean`。
- 用于执行动作，不代表真的生成一个同名文件。

## 模式规则

- 用通配形式简化重复写法。
