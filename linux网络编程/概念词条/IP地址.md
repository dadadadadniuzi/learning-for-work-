# IP地址

## 它是什么

- IP 地址用于在网络层标识一台主机或一个网络接口。
- IPv4 常写成点分十进制形式，例如 `127.0.0.1`、`192.168.1.10`。

## 在程序里的表示

- 人读的形式通常是字符串。
- 内核地址结构中使用二进制网络字节序形式。
- IPv4 地址在 `sockaddr_in` 中放在 `sin_addr` 字段里。

## 常见转换

- [[linux网络编程/函数笔记/地址转换/inet_pton|inet_pton]]：字符串 IP 转二进制网络地址。
- [[linux网络编程/函数笔记/地址转换/inet_ntop|inet_ntop]]：二进制网络地址转字符串 IP。

## 易错点

- 字符串 `"127.0.0.1"` 不能直接赋给 `sin_addr.s_addr`。
- `127.0.0.1` 是本机回环地址，只能连接同一台主机上的服务。
