# ip

## 命令作用

- `ip` 是 Linux 上查看和管理网络配置的常用命令。
- 它可以查看网卡、IP 地址、路由表、邻居表等信息。
- 在网络编程学习中，常用它确认本机 IP、网卡状态和路由是否正常。

## 常见写法

```bash
ip addr
ip a
ip link
ip route
```

## 参数说明

- `addr` / `a`：查看或管理网卡 IP 地址。
- `link`：查看或管理网络接口状态。
- `route`：查看或管理路由表。

## 常见用途

- 写 TCP/UDP 程序前，确认本机 IP 地址。
- 排查客户端连不上服务器时，确认网卡是否启用。
- 查看默认路由是否存在。
