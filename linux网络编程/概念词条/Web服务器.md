# Web服务器

## 它是什么

Web 服务器是接收 HTTP 请求并返回网页或其他资源的服务器程序。

## 怎么理解

从网络编程角度看，它本质上仍然是一个 socket 服务器：

- 接受 TCP 连接
- 读取请求
- 解析请求路径
- 返回 HTTP 响应
