# netstat

## 命令作用

- `netstat` 用于查看网络连接、监听端口、路由表和网络统计信息。
- 它在很多旧课程、旧系统和资料中仍然很常见。

## 常见写法

```bash
netstat -tan
netstat -tln
netstat -tulpn
netstat -tanp | grep 9527
```

## 参数说明

- `-t`：显示 TCP。
- `-u`：显示 UDP。
- `-l`：显示监听状态。
- `-n`：数字方式显示地址和端口。
- `-p`：显示进程信息。
- `-a`：显示所有 socket。

## 常见用途

- 查看服务器是否监听了指定端口。
- 观察 TCP 连接状态。
- 排查端口是否被占用。

## 注意点

- 新系统可能默认没有安装 `netstat`，可优先使用 [[linux网络编程/指令查询/命令卡/ss|ss]]。
- `-p` 查看进程信息时可能需要 root 权限。
