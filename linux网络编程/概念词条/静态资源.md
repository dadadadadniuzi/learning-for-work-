# 静态资源

## 它是什么

静态资源是服务器不需要动态计算、通常直接从文件系统读取就可以返回的资源。

## 常见例子

- HTML
- CSS
- JavaScript
- 图片

## 怎么理解

静态 Web 服务器的核心任务，就是把 URL 路径映射到文件，再把文件内容按 HTTP 响应返回。
