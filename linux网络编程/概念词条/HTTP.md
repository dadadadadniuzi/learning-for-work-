# HTTP

## 它是什么

HTTP 是应用层协议，全称 HyperText Transfer Protocol，用来规定浏览器和 Web 服务器之间如何组织请求与响应。

## 怎么理解

- TCP 负责把字节可靠送达。
- HTTP 负责规定这些字节在应用层应该长成什么格式。
- 浏览器访问网页、请求图片、提交表单时，常常都在使用 HTTP。

## 核心特点

- 请求-响应模型：客户端发请求，服务器回响应。
- 文本协议特征明显：课程里最常见的是直接读取请求文本、拼接响应文本。
- 无状态：协议层面默认不记住前一次请求的上下文。
