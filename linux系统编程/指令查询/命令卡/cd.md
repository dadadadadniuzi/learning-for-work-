# cd

> [!info] 功能
> 切换当前工作目录。

## 语法

- `cd path`
- `cd ..`
- `cd ~`
- `cd -`

## 输入参数

- `path`：目标目录路径，可以是绝对路径，也可以是相对路径。
- `..`：切换到上一级目录。
- `~`：切换到当前用户家目录。
- `-`：返回上一次所在目录。

## 输出

- 改变 shell 当前工作目录。

## 常见用法

- 进入项目目录。
- 在多个目录之间来回跳转。

## 易错点

- `cd` 是 shell 内建命令，不是普通程序。
