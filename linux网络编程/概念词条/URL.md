# URL

## 它是什么

URL 是统一资源定位符，用来描述浏览器要访问哪个资源。

## 常见结构

```text
协议://主机:端口/路径
```

例如：

```text
http://127.0.0.1:8080/index.html
```

## 怎么理解

- `http`：使用什么应用层协议。
- `127.0.0.1`：访问哪台主机。
- `8080`：访问主机上的哪个服务端口。
- `/index.html`：请求哪个资源路径。
