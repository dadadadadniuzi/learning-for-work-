# size_t

## 它是什么

- `size_t` 是 C 语言中用于表示对象大小、缓冲区长度、字节数的无符号整数类型。

## 怎么理解

- 它通常用来表示“有多大”或“最多处理多少字节”。
- 在 `send`、`recv` 里，`len` 参数使用 `size_t` 表示缓冲区长度或希望处理的字节数。

## 易错点

- `size_t` 是无符号类型，不适合用来表示可能为 `-1` 的返回值。
- 系统调用或网络函数返回实际读写字节数时，常用 [[linux网络编程/概念词条/ssize_t|ssize_t]]。
