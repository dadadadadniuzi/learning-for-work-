# 浏览器访问服务器流程

## 它是什么

浏览器访问服务器流程，是指用户输入 URL 后，浏览器和服务器之间从建立连接到页面展示的大致过程。

## 常见步骤

```text
输入 URL
  ↓
建立 TCP 连接
  ↓
发送 HTTP 请求
  ↓
服务器返回 HTTP 响应
  ↓
浏览器解析 HTML
  ↓
继续请求 CSS / JS / 图片等资源
```

## 怎么理解

网页通常不是一个单独文件，而是一组资源的集合。浏览器先拿到 HTML，再根据 HTML 里引用的资源继续发请求。
