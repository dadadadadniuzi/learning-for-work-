# HTTP方法

## 它是什么

HTTP 方法表示这次请求想对目标资源做什么操作。

## 常见方法

- `GET`：获取资源。课程里的静态 Web 服务器最常见。
- `POST`：提交数据。
- `HEAD`：只要响应头，不要响应体。

## 怎么理解

简单课程项目里，最先要掌握的是 `GET`，因为浏览器请求静态 HTML、CSS、图片时最常见的就是它。
