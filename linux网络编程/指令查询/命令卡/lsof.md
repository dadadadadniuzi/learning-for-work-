# lsof

## 命令作用

- `lsof` 用于列出进程打开的文件。
- 在 Linux 中 socket 也可以看作文件对象，因此 `lsof` 可以用来查看端口被哪个进程占用。

## 常见写法

```bash
lsof -i :9527
lsof -iTCP:9527 -sTCP:LISTEN
lsof -i
```

## 参数说明

- `-i`：显示网络相关文件。
- `:端口`：按端口过滤。
- `-iTCP:端口`：按 TCP 端口过滤。
- `-sTCP:LISTEN`：只看 TCP 监听状态。

## 常见用途

- 查找占用某个端口的进程。
- 配合 `kill` 结束误占端口的测试程序。
- 排查服务器重启时 `bind` 失败的原因。

## 示例

```bash
lsof -i :9527
```

这条命令用于查看哪个进程正在使用 `9527` 端口。
