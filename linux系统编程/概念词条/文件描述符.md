# 文件描述符

## 它是什么
- 进程打开文件后，内核返回的一个非负整数。
- 它是进程访问文件、管道、套接字等内核对象的入口编号。

## 怎么理解
- 你可以把它看成“内核帮你保管的资源编号”。
- `0`、`1`、`2` 分别是标准输入、标准输出、标准错误。
- `open` 返回的就是文件描述符。
