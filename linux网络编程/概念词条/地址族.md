# 地址族

## 它是什么

- 地址族用于说明 socket 使用哪一种地址格式和协议族。
- 在 `socket` 的 `domain` 参数和地址结构的 `sin_family` 字段中经常出现。

## 常见取值

- `AF_INET`：IPv4 网络通信。
- `AF_INET6`：IPv6 网络通信。
- `AF_UNIX` / `AF_LOCAL`：本地 Unix Domain Socket。

## 怎么理解

- 选择 `AF_INET` 后，地址结构通常使用 [[linux网络编程/概念词条/sockaddr_in|sockaddr_in]]。
- 选择 `AF_UNIX` 后，地址结构通常使用 [[linux网络编程/概念词条/sockaddr_un|sockaddr_un]]。

## 易错点

- `socket` 的 `domain` 和地址结构里的 `sin_family` 要匹配。
- IPv4 程序里不要把 `AF_INET` 和 `AF_UNIX` 混用。
