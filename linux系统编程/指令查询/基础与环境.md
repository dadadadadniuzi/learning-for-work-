## 环境查看

- `cat /etc/shells`：查看系统允许使用的 shell 列表。
- `echo $SHELL`：查看当前用户正在使用的 shell。

## 终端快捷键

- `Ctrl + p`：上一条历史命令
- `Ctrl + n`：下一条历史命令
- `Ctrl + b`：光标左移
- `Ctrl + f`：光标右移
- `Ctrl + a`：光标移到行首
- `Ctrl + e`：光标移到行尾
- `Ctrl + u`：删除光标前到行首
- `Ctrl + k`：删除光标后到行尾
- `Ctrl + l`：清屏
