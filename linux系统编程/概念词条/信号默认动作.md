# 信号默认动作

## 它是什么
- 内核为每种信号预设的一种处理结果。

## 常见默认动作
- 终止进程。
- 终止并产生 core 文件。
- 暂停进程。
- 忽略。
- 继续运行。

## 怎么理解
- 默认动作不是“程序写出来的”，而是“系统默认帮你做的”。
- 当没有自定义捕捉时，信号通常按默认动作走。
