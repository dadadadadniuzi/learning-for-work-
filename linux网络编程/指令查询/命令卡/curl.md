# curl

## 它是什么

`curl` 是常用的命令行 HTTP/HTTPS 请求工具，也能用于测试服务器返回内容。

## 常见用法

查看网页响应内容：

```bash
curl http://127.0.0.1:8080/
```

只看响应头：

```bash
curl -I http://127.0.0.1:8080/
```

## 在课程中的作用

- 快速测试自己写的 HTTP 服务器是否能返回正确响应。
- 验证状态码、响应头、页面内容。
