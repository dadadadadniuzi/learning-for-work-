# putchar

> [!info] 功能
> 向标准输出写出一个字符。

## 函数原型

- int putchar(int c);

## 依赖头文件

- `#include <stdio.h>`

## 输入参数

- `c`：要输出的字符，类型写成 `int` 是为了兼容字符常量和 `EOF` 这类特殊值。真正输出时，通常会把它按无符号字符来解释成一个字节并写到标准输出。

## 输出参数

- 无直接输出参数。

## 返回值

- 成功时返回写出的字符。
- 失败时返回 `EOF`，并可结合 [[linux系统编程/概念词条/errno|errno]] 查看错误原因。

## 知识点补充

- `putchar` 可以理解成“字符版的输出函数”，适合逐个字符输出。
- 它默认写到标准输出，也就是通常看到的终端。
- 和 `printf` 相比，它不负责格式化；和 [[linux系统编程/函数笔记/文件IO/write.md|write]] 相比，它属于标准 I/O 库接口，带有库缓冲语义。

## 常见用法

- 输出单个字符，例如换行符、分隔符、循环里的每个字符。
- 搭配 `getchar` 做最基础的字符输入输出实验。

## 易错点

- 不要把 `putchar` 当成字符串输出函数，它一次只处理一个字符。
- 如果标准输出被重定向，`putchar` 的输出位置也会跟着变化。
