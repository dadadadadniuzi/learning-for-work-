# gcc

> [!info] 功能
> 编译、汇编并链接 C 程序。

## 语法

- `gcc file.c -o app`
- `gcc -E file.c -o file.i`
- `gcc -S file.i -o file.s`
- `gcc -c file.s -o file.o`

## 输入参数

- `file.c`：源文件。
- `-o`：指定输出文件名。
- `-E`：只做预处理。
- `-S`：只到汇编文件。
- `-c`：只到目标文件。
- `-g`：加入调试信息。
- `-I`：添加头文件搜索路径。
- `-L`：添加库文件搜索路径。
- `-lxxx`：链接名为 `libxxx.so` 或 `libxxx.a` 的库。

## 输出

- 预处理文件、汇编文件、目标文件或可执行文件。

## 常见用法

- 编译 C 程序。
- 调试前加 `-g`。
- 链接库时加 `-L` 和 `-l`。

## 易错点

- `-g` 只是加入调试信息，不代表程序一定无优化问题。
- 预处理、编译、汇编、链接是不同阶段。
