# bzip2

> [!info] 功能
> 另一种压缩文件的工具。

## 语法

- `bzip2 file`
- `bunzip2 file.bz2`

## 输入参数

- `file`：要压缩的文件。

## 输出

- 生成 `.bz2` 文件，或解压回原文件。

## 常见用法

- 和 `tar` 联用得到 `.tar.bz2`。

## 易错点

- 还是以单文件压缩为主，目录通常先打包。
