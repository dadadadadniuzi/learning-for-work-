# 端口

## 它是什么

- 端口用于在一台主机上区分不同的网络进程或服务。
- IP 地址定位主机，端口定位主机上的具体服务。

## 在程序里的表示

- TCP/UDP 端口是 16 位整数。
- 在 `sockaddr_in.sin_port` 中必须使用网络字节序。
- 常见写法是 `addr.sin_port = htons(9527);`。

## 常见范围

- 0 到 65535。
- 0 到 1023 通常是知名端口，普通用户程序一般不随便占用。
- 自己练习时常使用 8000、8080、9527 等端口。

## 易错点

- 端口不是进程 ID。
- 同一个 IP、同一个协议、同一个端口通常不能被多个服务随意重复绑定，除非设置端口复用等选项。
