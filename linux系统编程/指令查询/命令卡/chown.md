# chown

> [!info] 功能
> 修改文件所有者或所属组。

## 语法

- `chown user file`
- `chown user:group file`

## 输入参数

- `user`：新的文件所有者。
- `group`：新的文件所属组。
- `file`：目标文件或目录。

## 输出

- 文件属主或属组被修改。

## 常见用法

- 把文件交给别的用户管理。

## 易错点

- 只有有权限的用户才能改所有者。
