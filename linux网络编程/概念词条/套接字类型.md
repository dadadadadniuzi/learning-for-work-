# 套接字类型

## 它是什么

- 套接字类型用于说明 socket 的传输语义。
- 它是 `socket` 函数的 `type` 参数。

## 常见取值

- `SOCK_STREAM`：字节流套接字，通常对应 [[linux网络编程/概念词条/TCP|TCP]]。
- `SOCK_DGRAM`：数据报套接字，通常对应 [[linux网络编程/概念词条/UDP|UDP]]。

## 怎么理解

- `SOCK_STREAM` 重视可靠、有序、面向连接。
- `SOCK_DGRAM` 重视轻量、保留数据报边界，但不保证可靠到达。

## 易错点

- `SOCK_STREAM` 不是“发送字符串”的意思，而是字节流传输语义。
- `SOCK_DGRAM` 不需要 `listen/accept` 这一套 TCP 监听连接流程。
