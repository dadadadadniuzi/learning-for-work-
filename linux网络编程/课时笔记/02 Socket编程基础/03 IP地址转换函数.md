# IP地址转换函数

## 本节学什么

- [[linux网络编程/概念词条/IP地址|IP地址]]在字符串和二进制之间的转换
- `inet_pton`
- `inet_ntop`
- `sockaddr_in.sin_addr` 的填写和读取

## 本节学什么详解

- [[linux网络编程/概念词条/IP地址|IP地址]]转换：程序里常看到 `"127.0.0.1"` 这种点分十进制字符串，但内核地址结构需要的是二进制网络字节序地址，因此需要转换函数。
- `inet_pton`：presentation to network，把可读字符串 IP 转成网络字节序二进制地址，常用于客户端填写服务器地址。
- `inet_ntop`：network to presentation，把网络字节序二进制地址转成可读字符串，常用于服务器打印客户端 IP。
- `sockaddr_in.sin_addr`：IPv4 地址结构中的地址字段，常通过 `inet_pton` 填写，或通过 `inet_ntop` 取出显示。

## 知识点补充

- `inet_pton` 和 `inet_ntop` 支持 IPv4 和 IPv6，具体由 `af` 参数决定。
- IPv4 常用 `AF_INET`，IPv6 常用 `AF_INET6`。
- 旧函数 `inet_addr`、`inet_ntoa` 有局限性，现代代码更推荐 `inet_pton` 和 `inet_ntop`。

## 本节内容速览

- 字符串 IP 给人看，二进制网络地址给内核用。
- `inet_pton`：字符串转网络地址。
- `inet_ntop`：网络地址转字符串。
