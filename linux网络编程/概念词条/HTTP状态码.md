# HTTP状态码

## 它是什么

HTTP 状态码表示服务器对这次请求的处理结果。

## 常见状态码

- `200 OK`：请求成功。
- `404 Not Found`：请求的资源不存在。
- `500 Internal Server Error`：服务器内部错误。

## 怎么理解

浏览器看到状态码后，就知道这次请求是成功了、资源没找到，还是服务器出错了。
