# mkdir

> [!info] 功能
> 创建目录。

## 语法

- `mkdir dir`
- `mkdir -p a/b/c`

## 输入参数

- `dir`：要创建的目录名或目录路径。
- `-p`：父目录不存在时自动创建父目录链。

## 输出

- 创建成功后得到一个新目录。

## 常见用法

- 新建项目目录。
- 递归创建多层目录结构。

## 易错点

- 默认不递归创建，父目录不存在会失败。
