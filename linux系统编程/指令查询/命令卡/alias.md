# alias

> [!info] 功能
> 给命令起别名。

## 语法

- `alias ll='ls -l'`
- `alias`

## 输入参数

- 别名名称和它要替代的命令。

## 输出

- 显示当前别名列表，或创建新的别名。

## 常见用法

- 简化高频命令。

## 易错点

- 只对当前 shell 或配置后的 shell 生效。
