# HTTP响应报文

## 它是什么

HTTP 响应报文是服务器返回给浏览器或客户端的应用层文本格式。

## 常见结构

```text
状态行
响应头
空行
响应体
```

## 状态行

常见形式：

```text
HTTP/1.1 200 OK
```

其中包含：

- 协议版本
- [[linux网络编程/概念词条/HTTP状态码|状态码]]
- 状态描述

## 怎么理解

课程里的简易 Web 服务器，至少要能返回：

- `200 OK`
- `404 Not Found`
