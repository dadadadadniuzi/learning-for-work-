# cp

> [!info] 功能
> 复制文件或目录。

## 语法

- `cp src dst`
- `cp -r dir1 dir2`
- `cp -a dir1 dir2`

## 输入参数

- `src`：源文件或源目录。
- `dst`：目标文件或目标目录。
- `-r`：递归复制目录。
- `-a`：尽量保留权限、时间、链接等属性。
- `-i`：覆盖前询问。

## 输出

- 在目标位置生成副本。

## 常见用法

- 复制文件备份。
- 复制目录树。

## 易错点

- 复制目录通常要加 `-r` 或 `-a`。
