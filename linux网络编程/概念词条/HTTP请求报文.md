# HTTP请求报文

## 它是什么

HTTP 请求报文是浏览器或客户端发给服务器的应用层文本格式。

## 常见结构

```text
请求行
请求头
空行
请求体
```

## 请求行

请求行最关键，常见形式：

```text
GET /index.html HTTP/1.1
```

它包含：

- 方法：如 `GET`
- 路径：如 `/index.html`
- 协议版本：如 `HTTP/1.1`

## 怎么理解

简化版静态服务器最少通常要从请求行里取出“方法”和“路径”，然后决定返回哪个文件。
